# 🧺 Task Basket

Task Basket is a full-stack productivity web application designed to help users organize, manage, and prioritize tasks efficiently through an intuitive and modern dashboard experience.

Users can create, edit, complete, and track tasks while also viewing real-time weather conditions and severe weather alerts based on their postal code to better plan their schedules.

Check out the website here: https://task-basket.onrender.com/

---

# ✨ Features

## 🔐 User Authentication
- Secure account registration and login using Firebase Authentication
- Protected routes and authenticated API requests
- JWT token verification using Firebase Admin SDK

## ✅ Task Management
- Create tasks with:
  - task names
  - descriptions
  - start dates
  - deadlines
  - locations
- Edit and delete tasks
- Mark tasks as completed
- Automatically categorize:
  - Due Today
  - Upcoming
  - Overdue
  - Completed

## 🌦 Weather Integration
- Real-time weather forecasts using WeatherAPI
- Weather condition icons and temperature data
- Severe weather alert detection
- Dynamic dashboard weather warning banners
- Postal-code-based weather lookup

## 🚦 Traffic Planning (In Progress)
- Planned future integration for traffic conditions and commute planning

## 📱 Responsive Design
- Mobile-friendly responsive layouts
- Bootstrap-powered UI components
- Modern dashboard design with alerts, cards, and interactive sections
